    About This Application

    Astro Escape is a digital board game currently in Alpha development. The core gameplay loop is fully implemented and playable: players take turns rolling dice, moving across the board, triggering tile effects, and using items to disrupt opponents.

    The board features multiple functional tile types including danger tiles, black holes, skip turns, reverse effects, and a finish tile that triggers victory. Strategic item usage, inventory management, and turn order create a competitive yet chaotic multiplayer experience.

    Matches are designed to last 10–20 minutes and support 2–4 players locally on the same device. While the prototype is fully playable, polish, sound effects, minigames, and visual upgrades are still in development.

    KEY CAPABILITIES

    The Alpha build includes a complete inventory system with 16 functional items across four rarity tiers: Common, Rare, Epic, and Legendary. Items can manipulate dice rolls, reposition players, alter board tiles, or reset progress entirely.

    Core systems implemented include: board generation, player movement with pathfinding, turn management, win conditions, UI systems, character persistence, and tile-triggered effects. All major gameplay systems are operational.

    CONTINUOUS UPDATES

    Astro Escape is currently in active Alpha development. The focus is on system polish, balance improvements, visual upgrades, and content expansion.

    Upcoming milestones include minigame implementation, sound effects, improved dice visuals, item animations, tutorial mode, and UI refinement. A beta-ready state is estimated within 2–4 months.
